# Jamstack Principles

- Pre-render as much as possible (HTML > JavaScript)
- CDNs, CDNs, CDNs
- Auto-scaling microservices
- Reduced attack surface

---

# Static Site Generators

- Pull data from various sources at build time
- Pre-render pages (also at build time)
- Build often, deploy often
  - Automated builds (via build hooks)
  - Atomic deploys
  - Instant cache invalidation
